Weather App

Live Demo: https://weather-app-yg.netlify.app/

Overview :

The Weather App is a user-friendly web application that allows users to fetch and display weather data based on their current location or by searching for a specific city. Utilizing the OpenWeatherMap API, this app provides real-time weather information, including temperature, humidity, wind speed.

Features :

Current Location Weather: Automatically fetches weather data based on the user's geolocation.
City Search: Allows users to search for weather information by entering a city name.
Session Storage: Stores user coordinates in session storage for quick access and improved user experience.
Responsive Design: Adapts to various screen sizes for an optimal user experience.
Error Handling: Displays user-friendly error messages for failed API calls or invalid input.

Technologies Used :

HTML
SCSS (CSS Preprocessor)
JavaScript
OpenWeatherMap API
Geolocation API
dotenv (for environment variables)

Usage :

Open the application in your web browser.
Grant location access when prompted, or enter a city name in the search bar.
The weather information will be displayed, including temperature, humidity, and weather conditions.
Coordinates are stored in session storage to streamline the experience, allowing for quick access to previously fetched user weather data.

API Reference :

The app fetches data from the OpenWeatherMap API. You can learn more about the API and its endpoints at OpenWeatherMap API Documentation.
